# SimpleHTTPServer_local_python3.x
支持python3.x，源码根据@xinghaixu做修改
基于内置http.server服务器完成轻量化的共享文件夹，前提是电脑手机在同一内网下。

## 操作流程
直接运行.py文件，这里设定的就是共享这个文件夹，还没更改文件夹的位置。
然后获取你电脑端的ipv4地址
**windows**使用ipconfig

**ios** 使用ifconfig
